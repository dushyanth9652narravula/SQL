# Number Functions

- SQL provides two number functions to manipulate integer or floating point data. Those are 

  **ROUND()** : ROUND() function is used to round the decimal point values to specifed number of decimals. The basic syntax of round is :

  `ROUND(<decimal>,no of decimals)`

  `ROUND(3.516,2) -> 3.52`

  `ROUND(3.516,1) -> 3.5`

  `ROUND(3.516,0) -> 4`


  **ABS()** : ABS() functions converts all negative values to positive values. The basic syntax is 

  `ABS(<value>)`

  `ABS(-10) -> 10`

  `ABS(20) -> 20`
